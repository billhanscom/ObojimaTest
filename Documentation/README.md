# Obojima Foraging Model Lab

Open `index.html` in a modern browser and choose **Run Full Analysis**. The application is self-contained: ingredient, Region, Search Area, and foraging configuration data are embedded locally. It does not contact a server and does not modify the Potion Almanac.

The Lab tests every valid Region–Search Area combination at DC 18 with a strong success (+5). Quick, Standard, and Deep depths run 2,000, 10,000, or 100,000 simulated searches per location.

Recommendations are diagnostic. A **Test this suggestion** button applies a temporary copy of the proposed adjustment and reruns the model. No source file is changed. The report button exports the current analysis as JSON.
