# Focused Search Testing

Use the Focused Search Simulator when you want to inspect one exact search rather than analyze the entire model.

1. Choose a Region and one of the Search Areas available there.
2. Choose the DC tier and Degree of Success.
3. Run 1,000, 10,000, or 100,000 searches.
4. Review how often each ingredient appeared and why it qualified.
5. Save additional tests and compare their rates per 1,000 searches.

The Model Workshop lets you change a Search Area's Civilization score and related Search Areas. New tests use the current workshop settings, while earlier saved runs retain the settings that produced them.

Nothing is written back to the Almanac automatically. Use the export controls to download adjusted data after testing.
