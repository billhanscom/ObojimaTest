# Obojima Foraging Model Lab

Open `index.html` in a modern browser. No server or internet connection is required.

The Lab tests every valid Region–Search Area combination and provides:

- an overall model-health assessment;
- a measured Civilization–Refinement relationship;
- plain-language recommendations ranked by likely impact;
- temporary before-and-after tests of suggested changes;
- direct editing of Search Area Civilization values and ingredient data;
- downloads of the adjusted complete data set, ingredient list, Search Area list, and configuration;
- a detailed JSON analysis report.

Changes made in the Lab remain local to the browser session until they are downloaded. The Lab does not modify the live Potion Almanac.

## Analysis depths

- Quick: 2,000 searches per location.
- Standard: 10,000 searches per location.
- Deep: 100,000 searches per location.

Quick is suitable for trying adjustments. Deep is best for a final review once the model is close to finished.
