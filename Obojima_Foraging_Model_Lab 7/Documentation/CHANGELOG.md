# Changelog

- Added model-wide regression analysis for Civilization and observed ingredient refinement.
- Added prioritized, high-impact recommendations.
- Added safe before-and-after testing of suggested Search Area changes.
- Added the ability to keep or discard tested changes.
- Added a Model Workshop for editing Search Area Civilization values, ingredient refinement, forageability, and Search Area assignments.
- Added a local change log.

## Focused Search Simulator

- Added focused tests for one Region and Search Area at a time.
- Added 1,000, 10,000, and 100,000-search trial sizes.
- Added exact DC-tier and Degree-of-Success controls.
- Added ingredient appearance counts, normalized rates, rarity, refinement, habitat fit, regional relationship, associated Search Areas, and ingredient Regions.
- Added JSON and CSV exports for individual tests.
- Added saved runs and normalized multi-run comparison tables.
- Added on-the-fly editing of a Search Area's Civilization score and related Search Areas.
